# URP 示例页面使用指南

## 访问地址

启动服务器后，可以通过以下地址访问示例页面：

- **登录页面**: http://localhost:3000/demo/login.html
- **注册页面**: http://localhost:3000/demo/register.html
- **用户面板**: http://localhost:3000/demo/dashboard.html
- **管理面板**: http://localhost:3000/demo/admin.html

## 默认账户

系统已预置一个管理员账户：

- **邮箱**: admin@example.com
- **密码**: admin123
- **角色**: SuperAdmin（拥有所有权限）

## 功能说明

### 登录页面
- 使用邮箱和密码登录
- 登录成功后自动跳转到用户面板
- Token 自动存储在 localStorage

### 注册页面
- 使用邮箱和密码注册新账户
- 密码长度至少 6 位
- 注册成功后自动分配 Guest 角色
- 自动登录并跳转到用户面板

### 用户面板
- 查看当前用户的账户信息
- 查看分配的角色
- 查看拥有的权限列表
- SuperAdmin 用户可以看到"管理面板"链接

### 管理面板（需要 SuperAdmin 权限）
- **用户管理**: 查看所有用户，冻结/激活用户，删除用户
- **角色管理**: 查看所有角色，编辑角色信息，删除角色
- **权限管理**: 查看所有权限点，编辑权限信息，删除权限
- 实时统计：总用户数、总角色数、总权限数

## 设计特点

- **黑白灰极简风格**: 精致极简主义，高对比度，锐利几何感
- **左右不对称布局**: 登录/注册页面采用 6:4 比例，左侧品牌展示，右侧表单
- **响应式设计**: 小屏幕自动切换为上下布局
- **流畅交互**: cubic-bezier 缓动动画，微妙的悬停效果和扫描线纹理

## 技术实现

- 纯 HTML + CSS + JavaScript
- Google Fonts (Syne + IBM Plex Mono)
- 原生 Fetch API 调用后端接口
- localStorage 存储认证 Token

## 注意事项

这些是**非生产级示例页面**，主要用于：
- 演示 URP API 的使用方式
- 快速测试权限管理功能
- 作为前端集成的参考示例

生产环境建议使用专业的前端框架（React/Vue/Angular）构建完整的管理界面。
